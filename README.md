# crime-dashboard-ukraine
# Злочинність у регіонах

Навчальний вебзастосунок для аналізу злочинності в регіонах України.

## Можливості
- фільтрація за регіоном
- фільтрація за категорією злочину
- фільтрація за датою
- таблиця даних
- графік динаміки
- індекс криміногенності
- карта злочинності

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
